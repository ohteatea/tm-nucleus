Slider

Knob Position

The lower bound and upper bound positions can be changed by setting the "left" value of #min-handle and #max-handle elements.

The active area of the slider bar should reflect the correct value based on knob position and this can be achieved by setting the correct "left" and "width" value of .slider__range--active element.

Example

<style type="text/css">
#min-handle {
left: 32%;
}
#max-handle {
left: 85%;
}
.slider__range--active {
left: 32%;
width: 53%;
}
</style>
