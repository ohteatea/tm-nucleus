Palette Customization

To apply custom colors, use the _palette.scss file as your basis by creating a copy of it in your local project folder. Open the local file and look for this:

// BEGIN CUSTOMIZATIONS

You can safely ignore/redefine/delete the variables above this comment. Our CSS components only reference the variables that are defined after the above line.

To include your custom palette in the compiled output, you'll need to create a custom import for Nucleus:

@import '[path/to/custom]/palette';
@import '[path/to/nucleus]/base/all-custom';
@import '[path/to/nucleus]/mixins/all';
@import '[path/to/nucleus]/components/all';
@import '[path/to/nucleus]/states/all';

Note that you will not have access to any of Nucleus's functions or mixins at this point in your custom palette file since it precedes any imports from Nucleus.
