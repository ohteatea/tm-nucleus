What is Nucleus?

Nucleus is a CSS component styleguide built on atomic design principles utilizing BEM syntax and SASS goodness. Its primary mission is to create a consistent visual and coding foundation upon which other projects can build their sites.

While source files are written in .scss, we also provide compiled and minified CSS files for consumption if you choose to forgo using SASS in your build process.

The project is publicly available on Github and as an NPM module.

Coding Style

This is a quick document outlining some of the SASS coding guidelines that are different from the tmol-web-spring project. A prettier version of this will come in the future. :)

  1. No usages of @extend.

  2. Do not use the parent selector shorthand (&) to compose class names.

      // bad
  .parent {
    ...

    &__child {
      ...
    }
  }

    Instead, explicitly write out any class names.

      // good
  .parent {
    ...
  }

  .parent__child {
    ...
  }

    The one exception is for pseudo-selectors.

      // this is okay
  .link {
    ...

    &:hover {
      ...
    }
  }

  3. No blank lines to separate stanzas anymore. Property declaration is still enforced, however.
